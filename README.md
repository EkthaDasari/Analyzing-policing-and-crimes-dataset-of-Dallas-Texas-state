# Analyzing-policing-and-crimes-dataset-of-Dallas-Texas-state

The main aim of this project is to analyze the policing and crimes of Dallas, Texas state in the year 2016. The outcome of this work provides insights about how the crime rate can be decreased and how if any racism is involved in police suspecting and arrest of subjects. Understanding the patterns of crime and the reasons behind these crime records is important in order take precautions or measures to reduce crime in coming near times.
